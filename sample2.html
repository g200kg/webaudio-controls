WebAudio-Knobs

non-square image is used with specifying width/height instead of diameter.

Integrated and enhanced by g200kg

Enhancement from webaudio-knob/slider/switch

  • Editable value display widget 'webaudio-param' is added that can link to other controls.
  • Support touch panel devices
  • Default knob/slider/switch image is embedded to use when no image is specified.
  • Image resizing to specified width/height.
  • Value changing in step of specified value by "step" attribute, independent from pointing device sensitivity.
  • WebAudio-Knob : Support non-square knob image when specified width/height instead of diameter.
  • WebAudio-Slider : Support horizontal slider by "direction" attribute.
  • WebAudio-Switch : 'Kick' and 'Radio' mode is added.
  • Common interface to Pointing devices
    • Shift + Drag : fine control (knob/slider).
    • Ctrl + Click : Reset to initial value.
  • Common interface to Javascript
    • setValue() function to set value and redraw
    • Attributes
      • width / height : widget size
      • value : current value. also used as an initial value.
      • step : value step for knob/slider. The value is always be rounded to multiple of "step".
      • enable : enable control by pointing devices (knob/slider/switch)
      • valuetip : display value-tip on the control (knob/slider)
      • sensitivity : sensitivity of the pointing device (knob/slider)

Attributes

webaudio-knob

  • src : url of the vertical stitched knob image.
  • value : the current value. also used as initial value of the control if specified.
  • defvalue : the default value that is used when ctrl+click. initial "value" is used instead if not specified.
  • min / max : min value and max value of the control.
  • step : value step of the control.
  • width / height : knob size.
  • diameter : knob size. this attribute can be used instead of width / height if the image is square.
  • enable : enable the pointing device control.
  • sprites : frames of the stitched knob image. note that this number is (number of frames) -1.
  • sensitivity : pointing device sensitivity. default is 1 that means min-max range correspond to 128px.
  • valuetip : enable the overlayed value-tip display. 0 or 1 (default=1)

webaudio-slider

  • src : url of the slider base image.
  • knobsrc : url of the knob part image
  • value : the current value. also used as initial value of the control if specified.
  • defvalue : the default value that is used when ctrl+click. initial "value" is used instead if not specified.
  • min / max : min value and max value of the control.
  • step : value step of the control.
  • width / height : slider size.
  • knobwidth / knobheight : knob part size. in default, minimum of width and height.
  • ditchLength : knob movable length. Default is (height - knobheight) or (width - knobwidth) depends on direction.
  • direction : slider direction. 'vert' or 'horz'.
  • enable : enable the pointing device control.
  • sensitivity : pointing device sensitivity. default is 1 that means min-max range correspond to ditchLength.
  • valuetip : enable the overlayed value-tip display. 0 or 1 (default=1)

webaudio-switch

  • src : url of the vertical stitched switch image.
  • value : the current value (0 or 1). also used as initial value of the control if specified.
  • defvalue : the default value that is used when ctrl+click. initial "value" is used instead if not specified.
  • width / height : switch size.
  • type : switch type. "toggle" (default) / "kick" or "radio".
  • group : used if the "type" is "radio". only one switch can be set to "1" in same group.
  • enable : enable the pointing device control.

webaudio-param

  • src : url of the background image. black rectangle is assigned if no image is specified. transparent if set to 'null'.
  • value : the current value. usually same as linked control.
  • width / height : param display size.
  • fontsize : font size.
  • color : font color.
  • link : specify the linked webaudio-knob/slider/switch by Id.
